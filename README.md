# UFC Live Stream

SEO-optimized landing page for UFC live streams with 8 language support.

## Setup

```bash
pnpm install
pnpm dev
```

## Config

- **Telegram Channel**: @ufc_live_show (configured)
- **Domain**: Update `https://yourdomain.com` in all files
- **Events**: Edit `/lib/events.ts` to update UFC events

## Deploy

```bash
pnpm build
pnpm start
```

Deploy to Vercel or your preferred hosting platform.
